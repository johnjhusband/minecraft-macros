# Minecraft Combat Macros

Combat macro configurations for Minecraft Java Edition on Mac, using the **Macro Keybinds** Fabric mod.

## Setup

### Prerequisites
- Minecraft Java Edition (Mac)
- [Fabric Loader](https://fabricmc.net/)
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Macro Keybinds](https://www.curseforge.com/minecraft/mc-mods/macrokeybinds)

### Installation
1. Install Fabric Loader for your Minecraft version
2. Download Fabric API and place in `~/.minecraft/mods/`
3. Download Macro Keybinds and place in `~/.minecraft/mods/`
4. Launch Minecraft with the Fabric profile
5. In-game: Options > Controls > Macro Keybinds to configure

## Macros

Macro configurations will be added to the `macros/` directory.
